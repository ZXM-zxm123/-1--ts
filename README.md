# -1--ts
使用 TypeScript 开发叠叠消游戏。画布底部为堆叠区（宽 8 列），彩色方块（5 种颜色）从顶部随机出现并自由落体。玩家控制左右移动方块，按空格加速下落。当同一列上出现连续 3 个相同颜色方块时，消除该组并给上方所有方块增加额外得分（每个 +10）。步数限制为总发射次数（初始 50 次）。目标分数达到后过关。障碍：炸弹块（落地时炸毁相邻列方块）、粘滞块（落地后暂时锁定该列 3 秒）。道具：加速器（加快方块下落速度）、颜色变换（随机改变当前方块颜色）、炸弹（消除任意指定列底部 3 个方块）。包含 5 个关卡，每关目标分数递增，下落速度加快。本地保存最高分和关卡。提供三套主题（糖果、金属、丛林）。
